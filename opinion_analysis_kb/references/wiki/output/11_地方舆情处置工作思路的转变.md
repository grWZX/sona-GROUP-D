## 内容




2026年4月7日，“浙江宣传”发布文章《舆情处置要“一事一清”》，阅读量迅速突破十万。依托良好传播效果，其于4月8日接续推出《舆情应对不能止于“摆事实”》，形成系列评论。

;


《舆情处置要“一事一清”》侧重从方法论层面阐述官方舆情处置原则，提出把事儿结清、把话讲清、把情绪出清、把隐患扫清四条具体建议。



;




相较而言，《舆情应对不能止于“摆事实”》观点更为鲜明犀利。文章直指部分官方通报情感淡漠、态度暧昧、回应姿态居高临下等问题，并结合具体案例展开点名批评。





在很多重大公共事件中，官方通报一经发出，总会引起网民极大关注。一些地方和部门因为通报表述不够严谨、存在疑点争议，甚至与公众认知产生偏差等原因，导致舆情反复，一波未平一波又起。在网友看来，通报的一字一句所反映出的都是有关部门对事件的认知。比如，在“女硕士走失被找回”事件中，当地在警情通报中介绍了找回的过程，却使用了“收留”一词，引起网友强烈不满，认为其掩盖了事情的严重性，并呼吁予以严查。


之江轩，公众号：浙江宣传舆情应对不能止于“摆事实”


与“女硕士走失被走回事件”相似的是，2025年，在“甘肃天水幼儿血铅异常事件”中，第一次回应由天水市联合调查组于7月8日发布。但该回应未能全面回应公众核心关切，反而引发关于检测数据真实性、相关人员行为动机等多方面舆论质疑。

;


面对公众关切，甘肃省于7月12日成立
省委省政府调查组开展提级调查，并于7月2
0日发布第二次通报，针对前期舆论质疑逐一作出回应。


不少地方在起草情况通报时，顾虑细节披露会引发次生舆情，便在关键信息上采用“春秋笔法”、含糊其辞甚至刻意回避。结果往往事与愿违，此类因首次通报处置不当、信息缺失而不得不启动提级调查、发布二次回应的情形，既损害了政府公信力，也造成了行政资源的重复消耗。


面对这种“越回避、谣言越易滋生”的困境，近几年舆情处置工作走在全国前列的成都选择主动出击。今年1月，成都市公安局在辖区内开展专项整治，依法处置了一批长期以“同性恋”标签对成都进行污名化炒作的网络账号；成都网信部门也联合重庆两江新区网信部门，对炮制涉成都等地“标题党”文章的重庆某公司依法进行约谈，及时遏制不良舆论传播。



;





这一处置思路的转变，让笔者想起2021年成都四十九中学生坠亡事件。当年5月9日，成都市四十九中一名学生坠楼身亡，事件迅速引发舆论热议。当地于5月11日发布通报，仅认定排除刑事案件可能，却未对公众关切的诸多疑点作出充分回应。当晚，大量市民聚集在四十九中校门口，对学生系自杀的结论表示质疑，并要求校方给出明确解释。





;




在舆情持续升级的态势下，川观新闻也发布了《四问成都49中一学生坠亡事件》的评论文章，聚焦公众疑点发声。
新华社记者于5月11日深夜连夜走访学校、教育部门、公安机关及学生家属等多方关键主体，对外公开了关
键监控片段、学生生前言行及心理状态等完整证据链，细致还原了事件完整经过，有效回应了社会关切。


记者从监控视频中看到，小林还在18时24分53秒时进入学校负一楼水泵房，并在水泵房内手持一把疑似刀具数次割左手腕，中间表现出垂头、摇脑、情绪低落。在水泵房，小林共计停留约12分钟，记者留意到其手腕上当时已出现明显伤口。
针对学校不让家属看监控视频的网络质疑，警方告诉记者，因为相关监控是涉案（事）件证据，案发后，公安机关第一时间进行了调取封存，所以家属向学校申请查看，未能在当晚看到。5月10日11时许，小林父亲、亲友及律师等一行3人到公安机关查看了全部监控视频。


公众号：川观新闻深度还原成都49中学生坠亡事件


;




从2021年5月这场舆论风波，到2021年7月“成都网络举报”公众号上线运营并持续更新，常态化开展辟谣及处置信息发布工作，二者之间似乎存在着微妙的关联。



;





“成都网络举报”公众号
是成都市网络信息应急指挥中心官方权威账号，统筹全市网络举报工作，负责受理、协助处置网民对成都市属地互联网不良与违法信息的举报。从日常发布来看，成都市网络信息应急指挥中心职能不仅包括大数据监测舆情、受理网民举报，更推动了跨部门联动的合作处置机制落地。




;


协同不畅、信息共享不足，是舆情处置中的常见堵点。笔者在高校舆情应对材料中，提出过构建两级协同机制的思路：在校内协同层面，建立由高校党委统一领导，宣传部牵头统筹协调，学生工作处、教务处、保卫处、信息中心等相关部门分工负责、密切配合的舆情管理联动机制，细化各部门职责清单与协作流程，明确舆情处置的首接责任与会商制度；在校地协同层面，推动高校与地方宣传、网信、公安、教育等部门建立常态化沟通协作机制，搭建校地舆情信息共享平台，打破“信息孤岛”。

这种协同处置的思路，也为涉企舆情应对提供了可借鉴的路径。企业可主动建立与网信、公安部门的常态化沟通渠道：一方面，第一时间梳理舆情线索，对IP集中、内容雷同、批量发布的异常账号特征进行证据固定，为后续调查提供支撑；另一方面，依托地
方网信部门的举报与处置机制，推动对网络“水军”账号的依法约谈与清理，遏制不实信息的传播链条。


2025年7月以来，理想汽车、华为鸿蒙智行、小米公司等车企就先后向烟台市公安局报警称，某知名网络平台出现大量涉及相关品牌新能源汽车质量、企业负责人的负面文章。


警方研判发现，发布这类文章的账号数以千计，不仅IP地址集中，发文时间、内容也较为集中。针对这些异常现象，警方经过数月侦查查明，网络“水军”团伙注册多家公司，利用MCN机构操纵数千个平台账号发布文章。相关内容呈现出高度的组织化特征：有的串联炒作、歪曲解读，对企业进行恶意诋毁；有的伪装身份、虚构场景，冒充消费者发布不实体验；有的搬运洗稿、批量炮制，将个别问题放大渲染。表述各异但指向相同的海量负面信息短时间内集中发布，瞬间占领网络空间。有受害企业表示，曾以为这是同行攻击或敲诈勒索，但警方查明，网络“水军”的牟利模式为利用大量账号发文的转发和评赞数量，获取平台支付的流量收益。


新华社记者，公众号：新华视点黑小米、黑理想、黑鸿蒙智行……他们图啥


尤其值得警惕的是，随着AI生成内容技术快速普及，网络黑灰产内容生产门槛持续降低，造谣文案、批量稿件的生成效率与伪装度将进一步提升，舆情监测与识别难度显著加大。面对日趋智能化、产业化的网络抹黑行为，企业仅靠自身提升监测能力已远远不够，必须将外部协同机制前置，主动与属地网信、公安部门建立常态化对接，才能有效抵御规模化、组织化的攻击。


;





各区委网信办积极探索、主动作为，将“自媒体”治理融入优化营商网络环境、未成年人网络保护等网络综合治理工作中，并依托各区建立的自媒体专委会区级分会不断拓展“自媒体”行业自律的领域。截至目前，各区级分会“自媒体”会员共644个，覆盖粉丝量近9.4亿。


公众号：网信上海清朗浦江丨规范行业秩序，凝聚自律共识上海召开“自媒体”行业自律大会


在筑牢负面舆情防线的同时，部分地方已开始探索正向舆论引导的路径，联动新媒体人士、网络达人，围绕产业发展、重大活动等主题，以更具传播力的叙事方式提升地方形象，在舆论场上筑牢正面声量基础。



归根结底，舆情应对的核心目标，远不止于平息一时风波、化解单次危机，更在于通过坦诚、透明、共情式的沟通表达，修复社会信任、凝聚社会共识、增进社会认同。



舆情一头连着政府治理效能，一头连着群众的利益诉求。各级领导干部既要做好权威信息的“发布者”，又要当好群众的知心人、贴心人，如此方能答好舆情应对这道“必答题”，更好地营造理性包容的网络舆论生态。


之江轩，公众号：浙江宣传舆情应对不能止于“摆事实”



无论是地方政府部门、高校等事业单位，还是企业等市场主体，在舆情应对中都应彻底摒弃捂、躲、拖的陈旧思维，坚决摒弃重处置轻沟通、重形式轻实效的敷衍心态。


将舆情应对融入日常治理、贯穿工作全程，既做好负面舆情的减法，坚决遏制造谣传谣、恶意炒作行为，也做好正向引导的加法，持续传递正能量、弘扬主旋律，才能不断提升舆情处置能力，持续营造清朗健康、理性包容、积极向上的网络舆论生态。
