# 舆情处置要“一事一清”

舆情是社情民意的晴雨表，有舆情很多时候是不可避免的。特别是进入移动互联时代，人工智能快速发展，舆情的发生发酵更加复杂多变，舆情处置进入hard模式，以致一些地方、部门谈舆色变、应对失据。

像有的事件发生后，官方对外通报“已成立调查组”或“正在调查”，却迟迟不见调查结果，甚至成了“断头新闻”；有的热点回应，含糊其辞、顾左右而言他，以致事情变成了事故，吸引全网“追剧”“破案”；还有的采取“鸵鸟战术”，寄希望于舆情自然降温，结果把话语权拱手相让，“你不说”就只能“被人说”。避免上述种种问题，舆情处置要做到“一事一清”，有头有尾、形成闭环。


;

图源：视觉中国

把事儿结清。舆情不是凭空发生的，背后一定有事情。出了事，就不能怕事、躲事，关键看处理的态度和结果，有没有沟通对话的诚意、解决问题的魄力，能否切实把群众利益维护好。

应对舆情，先得解决事情。如果实际问题没解决，无异于“开着水龙头拖地”，舆情是“拖”不干净的。有的地方、部门就因实际工作有瑕疵，舆情一再发酵，甚至被贴上标签，长期与负面新闻“捆绑”，严重损害了形象。

把话讲清。这是职责所在。众声喧哗时，人人都在等一个权威说法。党委政府掌握源头信息，主动发布、及时发布、准确发布，是应担之责。世上没有不透风的墙，如果一味“删封堵”“捂盖子”，就会把小事拖大、大事拖炸。


;

图源：视觉中国

把话说清，也是能力体现。技巧、手段、话术是次要的，关键在于讲明事实、表明态度、亮明举措；对于质疑焦点、专业术语、特定表述，还得及时回应、解疑释惑。既不能玩文字游戏、把人当傻子，也不能自说自话、以为别人都懂。某地曾发布“保持10天以上存粮”倡议，没有充分考虑社会语境、评估舆情风险，结果一次常规发布引发公众猜疑恐慌，让人以为“要打仗了”。

把情绪出清。“知屋漏者在宇下，知政失者在草野”，如今的“草野”主要在网上。“人争一口气”，很多舆情都夹杂着情绪，尤其医疗、教育、“一老一小”等民生领域，更是群众急难愁盼和情绪积累集中的地方。应对处置舆情，不能简单“就事论事”，要尊重“人之常情”，重视人文关怀、做到换位思考，这样才能和民意共情共振，获得公众理解支持。

处置舆情，功夫也在平时。“键对键”不能替代“面对面”，要扑下身子感知社会情绪，及时排忧解难，做到“民呼我应”“民呼我为”。有的地方处置舆情往往抱着侥幸心理，觉得只要眼下这波热度压下去、暂时过关就算了事。殊不知，互联网是有记忆的。民众的不满、疑惑等情绪并不会凭空消失，反而会在一次次敷衍处置中悄悄积压。等到下一次舆情再起，旧账新账很容易被一并翻出，过往的负面印象会被反复关联放大，最终透支的是一个地方的整体公信力。

把隐患扫清。解决问题、力挽狂澜是成绩，发现问题、防患未然也是成绩。如苏轼《晁错论》所言，对于“不测之忧”，如果坐观其变、视而不见，将造成无法挽回的局面。居安思危、未雨绸缪，才能有备无患、遇事不慌。

“针尖大的窟窿能透过斗大的风”。防微杜渐贵在一个“早”字，早识别、早预警、早发现、早处置，尽可能掌握主动。对于已发生的舆情，也不能头痛医头、脚痛医脚，而应以“解决一件事”带动“解决一类事”，努力让已经发生的“坏事”，成为发现问题、清除风险的契机。

有人说，舆情最怕串味儿、最怕联想、最怕“翻旧账”。一事一清，知易行难。经一事，清一事，不仅能进一寸、长一智，练就处置舆情、转危为机的过硬本领，也能在一次次坦诚回应、务实行动中，不断夯实信任基础，持续赢得群众理解与支持。
