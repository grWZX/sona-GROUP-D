## 内容



果然还有猛料！

原以为张凌赫会靠着《逐玉》成功上桌，但谁曾想，如今桌是没上一点，人直接成了舆论中心的靶子。

网友调侃他“六点打仗，四点起来化妆”，以解放军官媒“钧正平”为首的多
家官媒，接连出来点名怒批。


;



一天之内，各省官媒紧随其后直接连环点名，火力之密集，近两年罕见，但让人想不到的是，如今这件事情全面升级。

一个更细思极恐的猜测浮出水面，张凌赫可能只是被推出来挡枪的炮灰，真正的角力，还在水面之下.....


;




;



3月27日，钧正平工作室发了篇评论，标题很长，叫《古装剧里涂脂抹粉的“将军”，承载不起塑造阳刚之气的社会责任》。

全文不到800字，但字字诛心，称将军形象可以帅，但不能为了帅而失真，同一天，浙江宣传也下场了。


;



文章把矛头对准了整个行业的创作倾向：“古偶不该只有皮囊没有筋骨”，如果影视作品越来越以颜值为核心标准，那么人物的精神力量、品格乃至故事厚度，都可能被边缘化。

换句话说，就是‘价值观在向颜值观让步’，而且浙江宣传还点出了一个更深的担忧：这种“粉底液将军”式的审美，会潜移默化影响青少年。


;



从这些都可以看出，官媒的评价一针见血，古装剧的目标观众是年轻人，如果输出的都是严重违背历史语境、“一眼假”的故事，对青少年群体的会产生不良影响。

更别说随着《逐玉》的爆火，已经走出国门，不少国家的海外观众看了，或许在他们眼中，这就是中华文化的真实样貌。


;



3月31日，《人民日报》也加入战局，山师大副教授卞芸璐发文，给这种现象起了个名字，叫做“空心病”。

简单来说，就是每一幕都很好看，却拼不出一个合理完整的故事世界，这种“预制感”不是创作偷懒，而是整个产业在流量与商业压力下逐渐走向保守。


;



平台与制作方在选角时，往往倾向优先考虑自带流量的演员，将其视为稳定收视与数据的‘保险牌’。

粉丝更在意的是偶像的外型、CP火花与画面质感，而非剧情本身的创新与深度，长期下来，创作空间被挤压，古装偶像剧陷入循环。


;



三大官媒齐发声，这在娱乐圈并不常见，上一次如此密集的官媒批评，还是2021年整治“娘炮文化”和“饭圈乱象”。

有分析指出，这次“钧正平”平时只发军政大事，极少点评影视，此次出手不同寻常，也证明这事，上头真在意。


;



;


按理说，官媒都下场了，明星团队该知道轻重，但张凌赫的粉丝和团队的操作，让所有人看傻了眼。

3月27日，钧正平发文当天，不少粉丝开始硬刚，还有微博热搜上出现了“舆论场应容得下粉底液将军”的话题。



;





前一天官媒刚强调“话语权不是粉底液涂出来的”，后脚热搜就来了个反向输出，有网友直呼：这是在和官媒对着干！


随着两极分化的舆论，一些吃瓜群众也炸开了锅，但张凌赫始终对这件事情没有做出回应，而是发了一条告别剧宣。

原以为这件事情会随着热度慢慢降下去，毕竟娱乐圈的事情来得快，去得也快，但更让人细思极恐的事情发生了.....


;



;





4月1月，有知情人士在某社交平台表示，大事不妙了，要开研讨会，研讨“粉底液将军”对电视剧行业的审美的偏差了。

这话一出，舆论彻底升级了，也就是说，因为《逐玉》一部戏，可能牵连很多个人，而且以后再出现这样的古偶剧恐怕难上加难。



;


随着舆论的持续发酵，更多观众开始深扒《逐玉》的剧情细节，结果发现，这剧的问题，远不止妆造。

据网友整理，剧中添加了大量原著里没有的情节，其中最受争议的，是美化“溃坝放水”的桥段，给下令的将军加了为了所谓“大义”背负骂名的高光场面。


;



还有女主说男主是英雄，“至于屠城，敌人做得，为何他就做不得”，到了战场上，女主又鼓动身边的人当逃兵，教别人冒领军功。

更离谱的是战场恋爱桥段：前一秒女主还在英勇杀敌，后一秒男主出现救场，女主发现自己被骗，在战场上拔腿就跑。


;



;



;



这些争议点迅速发酵，不少网友开始呼吁下架整顿，认为这不只是审美问题，更涉及价值观导向，给观众灌输屠城、逃兵等思维，很容易影响青少年的价值观。

有网友甚至呼吁严查幕后班底和投资人，还有人联想到官媒批评后张凌赫团队的反常操作，怀疑背后另有隐情。


;



;



;



更诡异的是，就在研讨会消息传出的同时，《逐玉》全集资源遭泄露，热搜霸榜，营销号齐刷刷带节奏“泄露是大爆剧的宿命”，被网友质疑“全流程造假”。

华谊兄弟股价从1.59跌到1.55，桃厂股价触底，资本“赔了夫人又折兵”的戏码，让这起事件蒙上了更复杂的色彩。


;



;



有网友的猜测越来越大胆：会不会是有人故意制造“粉底液将军”话题，把舆论焦点集中在张凌赫的个人形象上。

甚至有网友怀疑，这背后有反华势力在推波助澜，毕竟一个“武安侯”的剧照，竟然敢能被泰国军方拿去印在征兵海报上。


;



;



截止到目前为止，关于这场古偶妆造的争议还在持续发酵中，能走到今天这一步，恐怕谁都没想到。

从一开始的“粉底液将军”，到如今的各大官媒怒批，再到粉丝出来硬刚，张凌赫确实被坑惨了，也成了炮灰。


;



;



就是不知道，如果相关部门真的开了研讨会，还会牵连多少人，到底又是谁躲在背后，让这件事情持续发酵.....

对此，你们怎么看呢？
